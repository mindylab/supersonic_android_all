Supertonic TTS is a high-performance, on-device text-to-speech (TTS) system for Android. It uses Supertonic ONNX models to provide high-quality speech synthesis entirely offline, ensuring privacy and availability without an internet connection.

Key Features:
- Offline Inference: Runs entirely on-device using ONNX Runtime.
- System Integration: Standard Android TTS provider, usable in any app.
- Native Rust Core: High-performance inference powered by a native Rust backend.
- Material Design 3: A modern and clean user interface built with Jetpack Compose.
- Audio Export: Save synthesized speech as WAV files.
- Lexicon Management: Define custom pronunciation rules for specific terms.
- History & Queue: Easily track and manage your synthesis requests.
- Multilingual Support: Support for various voices and styles.

Whether you need a standalone TTS player or a system-wide speech engine, Supertonic TTS provides a fast, reliable, and private solution.
